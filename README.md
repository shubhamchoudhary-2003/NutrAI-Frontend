## Front-End

1. For the execution of front-end clone this repository.
    ```sh
    git clone https://github.com/shubhamchoudhary-2003/NutrAI-Frontend
    cd NutrAI-Frontend
    ```
    
2. Install the npm packages.
   ```sh
   npm i
   ```
   
3. Run the front-end Next app
    ```sh
    npm run dev
    ```
